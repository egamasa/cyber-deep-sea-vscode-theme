# Cyber Deep Sea — テーマ設計ドキュメント

## コンセプト

「深海 × サイバー」をテーマにしたダークカラーテーマ。
深海の暗く冷たい水中にサイバー的な発光が浮かぶイメージ。
長時間のコーディングでも目が疲れにくいよう、高彩度のネオン色は避け、
落ち着いたシアン灰系をベースにしている。

---

## ファイル構成

```
cyber-deep-sea-vscode-theme/
├── themes/
│   └── cyber-deep-sea-color-theme.json  ← テーマ本体（JSONC形式、コメント付き）
├── .vscode/
│   ├── launch.json                      ← F5 でテーマプレビュー起動
│   └── settings.json                    ← Prettier 自動整形設定
├── icon.svg                             ← アイコン原本（SVG）
├── icon.png                             ← アイコン（256x256 PNG、SVGから生成）
├── package.json                         ← 拡張機能マニフェスト
├── LICENSE                              ← MIT ライセンス
├── README.md                            ← Marketplace / GitHub 向け説明
├── THEME_DESIGN.md                      ← 本ドキュメント（設計方針・パレット定義）
├── .gitignore                           ← node_modules, *.vsix を除外
└── .vscodeignore                        ← .vsix パッケージから除外するファイル
```

### テーマファイルの構造

`themes/cyber-deep-sea-color-theme.json` は JSONC 形式で、以下の2セクションで構成：
- `colors` — UI 要素の色（262 色）。カテゴリごとにコメントで区切っている
- `tokenColors` — シンタックスハイライト（36 ルール）

### 色の選定基準

1. **彩度は55%以下**（コード中で面積の大きいトークン）
2. **色相を広く分散**（赤、オレンジ、黄、緑、シアン、青、紫の7系統）
3. **隣接する色相は明度で差をつける**（例: 青 `#7c9cd8` とシアン `#74b0c8`）
4. **全色 WCAG AA 基準（コントラスト比 4.5:1 以上）をクリア**
5. **fontStyle で追加の区別**（italic: コメント/パラメータ、bold: 型/定数）

## UI色の設計ルール

### 階層の表現

背景色の明度差で奥行きを作る：
```
エディタ(最暗) → ガター → サイドバー/パネル → アクティブタブ → アクティビティバー → ステータスバー(最明)
```

### 状態の表現

不透明度のバリエーションで hover / active / selected を表現する：
- `40` — アクティブ選択
- `30` — フォーカス
- `25` — 非アクティブ選択
- `20` — ドロップ先、薄いハイライト

### Git / Diff の色

テーマのセマンティック色をそのまま使用（不透明度100%で視認性を確保）：
- 追加: `#48d8a0`（アクセント緑）
- 変更: `#78c8e8`（アクセントシアン）
- 削除: `#d87878`（エラー赤）
- リネーム: `#8cc8c4`（シンボルティール）
- 未追跡: `#7cc098`（文字列緑）
- コンフリクト: `#d8a060`（警告アンバー）
- 無視: `#7c9cd850`（補助テキスト半透明）
- ステージ済み: 各色の `c0`（75%）不透明度

### ターミナル ANSI 色

通常色はトークンカラーを流用し、Bright 系は少し明るくした派生色を使用。

### インデントガイド / ブラケット レインボー

インデントガイドとブラケットハイライトで共通の6色を使用。
テーマのトークンカラーから抽出したレインボー配色。

| # | 色 | Hex | 元のトークン |
|---|---|---|---|
| 1 | 赤 | `#d87878` | キーワード |
| 2 | オレンジ | `#d8a878` | パラメータ |
| 3 | 黄 | `#c8c088` | 数値 |
| 4 | 緑 | `#7cc098` | 文字列 |
| 5 | シアン | `#78c8e8` | アクセントシアン |
| 6 | 紫 | `#c488d8` | メソッド定義 |

- インデントガイド通常: 不透明度 `40`（25%）
- インデントガイドアクティブ: 不透明度 `a0`（63%）
- ブラケット: 不透明度 `ff`（100%）

## 修正・改善時のガイドライン

> **必須ルール**: テーマの色を修正・変更した場合は、このドキュメントも必ず同時に更新すること。
> 色の追加・変更時はパレット表とトークンカラー表を、設計方針の変更時はガイドラインセクションを更新する。

### 新しい色を追加する場合

1. まず既存パレットの色で対応できないか検討する
2. 不透明度のバリエーションで解決できないか検討する
3. 新色が必要な場合は、彩度55%以下、背景とのコントラスト比4.5:1以上を守る

### 特定の色を変更する場合

1. その色が他の箇所でも使われていないか `grep` で確認する
2. 変更後に WCAG AA コントラスト比を確認する
3. 隣接する色相との区別が保たれているか確認する

### 確認用コマンド

```bash
# 特定の色の使用箇所を検索
grep -n '#48d8a0' themes/cyber-deep-sea-color-theme.json

# JSONC 構文チェック
node -e "
const fs = require('fs');
let c = fs.readFileSync('themes/cyber-deep-sea-color-theme.json','utf8');
let lines = c.split('\n').map(l => {
  let s=false,e=false;
  for(let i=0;i<l.length;i++){if(e){e=false;continue}if(l[i]==='\\\\'){e=true;continue}if(l[i]==='\"'){s=!s;continue}if(!s&&l[i]==='/'&&l[i+1]==='/') return l.substring(0,i);}
  return l;
}).join('\n').replace(/,(\s*[}\]])/g,'\$1');
try{const o=JSON.parse(lines);console.log('OK. Colors:',Object.keys(o.colors).length)}catch(e){console.error(e.message)}
"
```

## アイコン

### 仕様

- サイズ: 256x256px（Marketplace 要件は 128x128 以上）
- 形式: PNG（SVG から生成）
- 角丸: 24px

### デザイン要素

- 背景: テーマの背景色グラデーション（`#0f1c24` → `#0a1218`）
- ブラケット `{ }`: アクセントシアン `#78c8e8`、パスで描画（stroke-width: 12）
- 中央の光点: アクセント緑 `#48d8a0`（カーソル色）、深海の生物発光のイメージ
- 波線: ティール `#4a7a6e`、深海の潮流
- 浮遊パーティクル: 各トークンカラーの小さな円、深海のプランクトン発光

### PNG 生成方法

```bash
python3 -c "
import cairosvg
cairosvg.svg2png(url='icon.svg', write_to='icon.png', output_width=256, output_height=256)
"
```
